# Chrome Web Store 개인정보 보호 입력 가이드

현재 `manifest.json`과 개인정보처리방침 기준입니다. 아래 문구를 Chrome Web Store Developer Dashboard의 `개인정보 보호` 탭에 입력합니다.

## 전용 목적 설명

```text
사용자가 브라우저에 이미 로그인한 ChatGPT, Claude, Gemini, Grok 계정의 사용량, 잔여 횟수와 리셋 시점을 팝업·페이지 위젯·툴바 배지로 표시하고 임계치·소진·리셋을 로컬 알림으로 안내하는 단일 목적의 확장 프로그램입니다.
```

## 권한 요청 이유

### storage 사용 근거

```text
각 서비스에서 직접 조회한 사용량·잔여 횟수·리셋 시각 캐시, Claude 조직 선택 정보, Gemini 요청 보조 토큰 및 중복 알림 방지 상태를 chrome.storage.local에 저장합니다. 데이터는 개발자 서버로 전송되지 않습니다.
```

### alarms 사용 근거

```text
사용자가 확장 프로그램을 열지 않아도 설정된 간격으로 사용량을 갱신하고 배지와 알림 상태를 최신으로 유지하기 위해 사용합니다.
```

### notifications 사용 근거

```text
사용량 80%·100% 도달, 잔여 횟수 소진 및 한도 리셋을 Chrome의 로컬 알림으로 사용자에게 안내하기 위해 사용합니다.
```

### 호스트 권한 사용 근거

```text
claude.ai, chatgpt.com, gemini.google.com, grok.com에 로그인된 사용자의 사용량·한도·리셋 정보를 해당 서비스의 HTTPS 엔드포인트에서 직접 조회하고 각 서비스 페이지에 사용량 위젯을 표시하기 위해 사용합니다. 그 밖의 사이트에는 접근하지 않습니다.
```

## 원격 코드

`아니요, 원격 코드 권한을 사용하고 있지 않습니다.`를 선택합니다.

외부 서비스의 API 응답, HTML 또는 protobuf 응답은 데이터일 뿐 실행 코드가 아닙니다. 모든 JavaScript와 CSS는 업로드 ZIP 안에 포함되어 있으며 `eval`, 외부 `<script>`, 원격 JavaScript 또는 원격 Wasm을 사용하지 않습니다.

## 사용자 데이터 사용

Chrome 정책은 개발자 서버로 보내지 않고 기기에서만 처리하는 데이터도 공개 대상으로 봅니다. 과소 신고를 피하기 위해 다음 네 항목을 선택합니다.

- [x] 개인 식별 정보 — Claude 조직 식별자와 표시 이름을 로컬에서 처리할 수 있음
- [x] 인증 정보 — 기존 로그인 세션, 쿠키와 요청용 토큰을 필요한 요청에 한해 처리함
- [x] 사용자 활동 — AI 서비스 사용량, 잔여 횟수와 리셋 상태를 처리함
- [x] 웹사이트 콘텐츠 — 지원 서비스의 사용량 엔드포인트 응답과 요청 보조 값을 처리함

다음 항목은 선택하지 않습니다.

- [ ] 건강 정보
- [ ] 금융 및 결제 정보
- [ ] 개인적인 커뮤니케이션
- [ ] 위치
- [ ] 웹 기록

## 데이터 사용 인증

화면 아래의 세 가지 인증은 모두 체크합니다.

- [x] 승인된 사용 사례를 제외하고 사용자 데이터를 제3자에게 판매 또는 전송하지 않음
- [x] 전용 목적과 관련 없는 목적으로 사용자 데이터를 사용하거나 전송하지 않음
- [x] 신용도 판단 또는 대출을 위해 사용자 데이터를 사용하거나 전송하지 않음

## 개인정보처리방침 URL

```text
https://techkwon.github.io/ai-usage-limit-alerts/privacy/
```

페이지 공개가 확인된 뒤 URL을 입력하고 `임시저장`을 누릅니다. 개인정보처리방침, 대시보드 공개 내용과 실제 확장 프로그램 동작은 항상 일치해야 합니다.
