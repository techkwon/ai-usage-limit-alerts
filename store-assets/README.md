# Store Assets

This folder holds the text and release checklist used to submit `AI 사용량 한도 알림`.

## Files

| File | Purpose |
| --- | --- |
| `STORE_LISTING.md` | Ready-to-use Korean and English listing copy, search terms, and non-affiliation notice. |
| `SUBMISSION_GUIDE.md` | Current official submission requirements and direct documentation URLs for Chrome, Edge, and Whale. |
| `PRIVACY_POLICY.md` | Code-factual privacy policy draft based on the extension's current data flow. |
| `REVIEW_NOTES.md` | Reviewer guidance and the test sequence for store review. |
| `RELEASE_CHECKLIST.md` | Final pre-release checklist for packaging and submission. |
| `SCREENSHOT_PLAN.md` | Store screenshot order, captions, and privacy-safe capture rules. |
| `screenshots/ko/`, `screenshots/en/` | Four verified `1280x800` screenshots per locale, covering popup and page widgets in light and dark themes. |
| `promotional/ko/`, `promotional/en/` | Localized `440x280` small and `1400x560` large promotional tiles. |
| `icon300.png` | Edge listing logo derived from the approved extension icon. |
| `icon-size-preview.png` | Visual QA sheet for the 16, 48, and 128 px icon exports. |
| `icon-source/` | Generated source and transparent master retained outside the release ZIP. |
| `icon-archive/` | Previous icon exports retained for rollback and comparison. |

## Scope

- Keep the copy factual.
- Avoid superlatives such as `#1`, `best`, or similar keyword spam.
- Update these files when the extension's data flow, assets, or store requirements change.
- The HTML harness files are capture-only sources and are excluded from the release ZIP.
