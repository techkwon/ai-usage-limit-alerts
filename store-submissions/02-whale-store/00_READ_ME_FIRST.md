# NAVER Whale Store 등록 순서

## 1. 패키지

`package/ai-usage-limit-alerts-v1.0.0.zip`을 Whale Store 개발자 화면에 업로드합니다.

## 2. 등록 정보

- 기본 한국어 제목: `AI 사용량 한도 알림`
- 영어 등록란이 제공되면 제목 `AI Usage Limit Alerts`와 영어 설명을 함께 등록합니다.
- 두 언어의 문구는 `documents/STORE_LISTING.md`에서 복사합니다.
- 핵심 문구는 `완전 무료 · 광고 없음 · 구독 없음` / `Completely free · No ads · No subscription`입니다.

## 3. 이미지

- 아이콘: `images/icon-128x128.png`
- 한국어 스크린샷: `images/ko/01`부터 `04`까지
- 영어 현지화 등록란이 제공되면: `images/en/01`부터 `04`까지

Whale 제출용 이미지에는 프로모션 타일을 넣지 않았습니다. 현재 준비한 네 장은 모두 `1280x800`입니다.

## 4. 심사·공개

- 개인정보처리방침: `documents/PRIVACY_POLICY.md`
- 심사 안내: `documents/REVIEW_NOTES.md`
- 공개 범위, 언어, 제공 지역은 실제 Whale 등록 화면에서 월드 타깃에 맞게 선택합니다.
- 지원 이메일과 공개 개인정보처리방침 URL을 제출자가 입력합니다.

최종 제출 직전 `documents/RELEASE_CHECKLIST.md`를 확인합니다.
