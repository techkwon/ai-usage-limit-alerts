# 스토어 등록 파일 안내

세 스토어에 바로 제출할 수 있도록 패키지, 이미지, 문서를 분리했습니다.

## 제품 표기

| 언어 | 제목 | 짧은 이름 | 핵심 문구 |
| --- | --- | --- | --- |
| 한국어 | `AI 사용량 한도 알림` | `AI 한도 알림` | `완전 무료 · 광고 없음 · 구독 없음` |
| 영어 | `AI Usage Limit Alerts` | `AI Limits` | `Completely free · No ads · No subscription` |

브라우저 UI 언어가 `ko`로 시작하면 한국어를 표시하고, 그 외 언어에서는 영어를 표시합니다. 팝업과 페이지 위젯은 브라우저·운영체제의 라이트/다크 테마를 자동으로 따릅니다.

## 폴더

1. `01-chrome-web-store/` — Chrome Web Store
2. `02-whale-store/` — NAVER Whale Store
3. `03-edge-addons/` — Microsoft Edge Add-ons

각 폴더의 `00_READ_ME_FIRST.md`를 먼저 읽고 다음 하위 폴더를 사용합니다.

- `package/` — 실제 업로드 ZIP
- `images/` — 아이콘과 `ko`/`en` 언어별 이미지
- `documents/` — 등록 문구, 개인정보처리방침, 심사 메모, 체크리스트

## 현재 검증 상태

- ZIP의 루트에 `manifest.json`이 있으며 `_locales/en`, `_locales/ko`, 공용 국제화 파일이 포함되어 있습니다.
- 개발용 `tests/`, `store-assets/`, `scripts/`, `.omx/`는 ZIP에서 제외되었습니다.
- 기능·보안·HTML 안전성·국제화·테마 회귀 테스트 `26/26`을 통과했습니다.
- 스크린샷은 언어별 4장, 모두 `1280x800`입니다.
- Edge용 프로모션 이미지는 언어별 `440x280`, `1400x560`입니다. Chrome은 프로모션 타일을 현지화할 수 없어 월드 타깃 공용 영문 타일 한 세트를 사용합니다.

## 제출 전 반드시 입력할 사용자 정보

- 공개 지원 이메일
- 공개 개인정보처리방침 URL
- 개발자·게시자 표시 이름과 계정 정보
- 공개 범위와 국가·마켓 선택
- Chrome 등록 화면에서 프로모션 동영상 필드가 실제로 필수인지 확인하고, 필요할 경우 YouTube URL
- 유료 플랜 재현에 심사용 계정이 꼭 필요한 경우에만 스토어의 보호된 입력란에 임시 자격 증명 제공

비밀번호, 세션 쿠키, 토큰, API 키는 문서나 ZIP에 넣지 않습니다.
