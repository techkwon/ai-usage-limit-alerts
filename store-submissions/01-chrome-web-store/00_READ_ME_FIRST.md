# Chrome Web Store 등록 순서

## 1. 패키지

`package/ai-usage-limit-alerts-v1.0.0.zip`을 Chrome Web Store 개발자 대시보드에 업로드합니다.

## 2. 언어별 등록 정보

- 한국어(`ko`): 제목 `AI 사용량 한도 알림`
- 영어(`en`): 제목 `AI Usage Limit Alerts`
- 두 언어의 상세 문구는 `documents/STORE_LISTING.md`에서 복사합니다.
- 핵심 무료 문구를 첫 문단에 유지합니다.

## 3. 이미지

- 공용 아이콘: `images/icon-128x128.png`
- 한국어 스크린샷: `images/ko/01`부터 `04`까지 순서대로
- 영어 스크린샷: `images/en/01`부터 `04`까지 순서대로
- 월드 타깃 공용 작은 프로모션 타일: `images/global/small-promo-440x280.png`
- 월드 타깃 공용 대형 프로모션 타일: `images/global/large-promo-1400x560.png` — 대시보드에서 표시되는 경우 사용

Chrome은 스크린샷과 설명은 언어별로 등록할 수 있지만 프로모션 타일은 현지화할 수 없습니다. 따라서 공용 타일은 영어판을 사용합니다.

## 4. 개인정보·심사

- 개인정보처리방침 본문: `documents/PRIVACY_POLICY.md`
- 심사자 안내: `documents/REVIEW_NOTES.md`
- 공개 개인정보처리방침 URL과 지원 이메일은 제출자가 직접 입력해야 합니다.
- 권한 목적은 사용량 조회, 로컬 저장, 주기 갱신, 로컬 알림으로 한정해 설명합니다.

## 5. 배포

- 월드 타깃이면 공개 배포와 제공 가능한 전체 지역을 선택합니다.
- 한국어와 영어 현지화 등록을 모두 완료합니다.
- 실제 제출 화면에서 프로모션 동영상 필드가 필수인지 확인합니다. 필수라면 광고가 꺼진 YouTube 영상 URL을 입력합니다.

최종 제출 직전 `documents/RELEASE_CHECKLIST.md`를 확인합니다.
