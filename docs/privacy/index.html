AI 사용량 한도 알림 개인정보처리방침

본 방침은 확장 프로그램이 현재 코드 기준으로 사용자 데이터를 어떻게 접근·사용·저장·전송하는지 설명합니다.

1. 단일 목적

사용자가 브라우저에서 이미 로그인한 ChatGPT, Claude, Gemini, Grok 계정의 사용량, 잔여 횟수와 리셋 시점을 표시하고 로컬 브라우저 알림을 제공하는 것이 확장 프로그램의 유일한 목적입니다.

2. 처리하는 데이터

사용량 정보사용량 퍼센트, 잔여 횟수, 한도 종류, 리셋 시각과 조회 시각
인증 관련 정보기존 로그인 세션, 요청용 토큰과 쿠키를 필요한 요청에 한해 처리
계정 보조 정보Claude 조직 식별자·표시 이름·요금제 구분과 Gemini 요청용 토큰
알림 상태동일 임계치 알림의 반복을 막기 위한 로컬 상태

ChatGPT 액세스 토큰은 메모리에서 일시적으로만 사용하며 영구 저장하지 않습니다. Gemini 요청용 토큰과 Claude 조직 선택 정보는 효율적인 갱신을 위해 chrome.storage.local에 저장될 수 있습니다. 확장 프로그램은 비밀번호를 읽지 않습니다.

3. 로컬 저장

사용량 캐시, Claude 조직 선택 정보, Gemini 요청용 토큰과 알림 중복 방지 상태는 사용자의 Chrome 브라우저 프로필에만 저장됩니다. 새 조회 결과로 덮어쓰며, 사용자가 확장 프로그램 데이터를 삭제하거나 확장 프로그램을 제거하면 삭제할 수 있습니다.

4. 네트워크 전송

확장 프로그램은 기능 제공에 필요한 경우에만 HTTPS를 통해 claude.ai, chatgpt.com, gemini.google.com, grok.com에 직접 요청합니다. 사용자 데이터는 개발자가 운영하는 서버나 무관한 제3자에게 전송되지 않습니다.

5. 수집하지 않는 데이터

  • 채팅 내용, 프롬프트와 개인 커뮤니케이션
  • 비밀번호, 복구 코드, 금융 및 결제 정보
  • 건강 정보, 위치, 전체 웹 방문 기록
  • 광고 식별자, 개발자 분석 이벤트, 원격 텔레메트리

6. 권한 사용

  • storage: 사용량 캐시, 요청 보조 값과 알림 상태를 기기에 저장
  • alarms: 일정한 간격으로 사용량을 갱신
  • notifications: 한도 임계치, 소진과 리셋을 로컬 알림으로 표시
  • 호스트 권한: 네 AI 서비스의 사용량 엔드포인트에 직접 요청하고 해당 페이지에 사용량 위젯을 표시

7. 판매·광고·사람의 열람 금지

개발자는 사용자 데이터를 수집해 판매하거나 광고, 신용 평가, 대출 또는 관련 없는 목적으로 사용하지 않습니다. 개발자 서버에 사용자 데이터가 전송되지 않으므로 개발자나 다른 사람이 해당 데이터를 열람하지 않습니다.

Chrome Web Store Limited Use 준수
확장 프로그램이 처리하는 데이터는 공개된 단일 목적 기능을 제공하는 데에만 사용됩니다. Google API에서 받은 정보의 사용은 Limited Use 요구사항을 포함한 Chrome Web Store 사용자 데이터 정책을 준수합니다.

8. 문의와 변경

문의 또는 삭제 관련 도움은 GitHub Issues를 이용해 주세요. 데이터 처리 방식이 바뀌면 배포 전에 본 방침과 Chrome Web Store 공개 내용을 함께 갱신합니다.

AI Usage Limit Alerts Privacy Policy

This policy explains how the extension accesses, uses, stores, and transmits user data based on its current implementation.

1. Single purpose

The extension's sole purpose is to display usage, remaining counts, and reset times for ChatGPT, Claude, Gemini, and Grok accounts already signed in through the browser, and to provide local browser notifications.

2. Data handled

Usage informationUsage percentages, remaining counts, limit types, reset times, and retrieval times
Authentication-related dataExisting sign-in sessions, request tokens, and cookies used only for required requests
Account helper dataClaude organization identifiers, display names, plan types, and a Gemini request token
Notification stateLocal state used to prevent duplicate threshold notifications

A ChatGPT access token is used transiently in memory and is not persisted. A Gemini request token and Claude organization selection may be stored in chrome.storage.local for efficient refreshes. The extension does not read passwords.

3. Local storage

Usage caches, Claude organization selection, a Gemini request token, and notification state are stored only in the user's Chrome browser profile. Cached values are replaced by newer results and can be removed by clearing extension data or uninstalling the extension.

4. Network transmission

Only when needed for the user-facing feature, the extension makes direct HTTPS requests to claude.ai, chatgpt.com, gemini.google.com, and grok.com. User data is not sent to a developer-operated server or to unrelated third parties.

5. Data not collected

  • Chat content, prompts, or personal communications
  • Passwords, recovery codes, financial or payment information
  • Health information, location, or general browsing history
  • Advertising identifiers, developer analytics, or remote telemetry

6. Permission use

  • storage: store usage caches, request helper values, and notification state locally
  • alarms: refresh usage at scheduled intervals
  • notifications: show local alerts for thresholds, exhaustion, and resets
  • Host permissions: request usage endpoints directly and display usage widgets on the four supported AI service pages

7. No sale, advertising, or human access

The developer does not collect user data for sale, advertising, credit decisions, lending, or unrelated purposes. Because user data is not sent to a developer-operated server, neither the developer nor other humans can access it.

Chrome Web Store Limited Use compliance
Data handled by the extension is used only to provide its disclosed single-purpose features. The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

8. Contact and changes

For privacy questions or deletion assistance, contact the developer through GitHub Issues. If the extension's data practices change, this policy and the Chrome Web Store disclosures will be updated before release.
