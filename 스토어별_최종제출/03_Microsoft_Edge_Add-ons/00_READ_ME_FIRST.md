# Microsoft Edge Add-ons 등록 순서

## 1. 패키지

`package/ai-usage-limit-alerts-v1.0.0.zip`을 Microsoft Partner Center에 업로드합니다.

## 2. 언어별 Store listing

- 한국어와 영어를 각각 추가합니다.
- 한국어 제목: `AI 사용량 한도 알림`
- 영어 제목: `AI Usage Limit Alerts`
- 각 언어의 상세 설명과 검색어는 `documents/STORE_LISTING.md`에서 복사합니다.

패키지의 `__MSG_*__`, `default_locale: en`, `_locales/ko`, `_locales/en` 구성이 두 언어를 식별하도록 준비되어 있습니다.

## 3. 이미지

- 확장 로고: `images/icon-128x128.png`
- 보조 고해상도 로고: `images/icon-300x300.png`
- 한국어: `images/ko/`의 스크린샷 4장과 두 프로모션 타일
- 영어: `images/en/`의 스크린샷 4장과 두 프로모션 타일

각 언어의 작은 타일은 `440x280`, 대형 타일은 `1400x560`, 스크린샷은 `1280x800`입니다.

## 4. 개인정보·심사

- 개인정보처리방침: `documents/PRIVACY_POLICY.md`
- 심사 메모: `documents/REVIEW_NOTES.md`
- 공개 개인정보처리방침 URL과 지원 이메일은 제출자가 입력합니다.
- Store listing의 언어별 설명은 해당 언어로 완성합니다.

## 5. Availability

- 월드 타깃이면 `Public`과 전체 마켓을 선택합니다.
- 한국어와 영어 Store listing을 모두 완료한 뒤 제출합니다.
- 선택 사항인 YouTube 영상은 준비된 영상이 있을 때만 등록합니다.

최종 제출 직전 `documents/RELEASE_CHECKLIST.md`를 확인합니다.
