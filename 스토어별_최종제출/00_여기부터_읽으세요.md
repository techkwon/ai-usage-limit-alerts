# 스토어별 최종 제출 파일

스토어 등록 시에는 이 폴더만 사용합니다. Chrome, Whale, Edge가 서로 섞이지 않도록 각 스토어 폴더 안에 업로드 ZIP, 이미지, 등록 문서를 모두 복사해 두었습니다.

## 1. Chrome Web Store

`01_Chrome_Web_Store/`

- `package/` — Chrome 대시보드에 업로드할 ZIP 1개
- `images/` — 128px 아이콘, 한국어·영어 스크린샷, 월드 타깃 공용 영문 프로모션 타일
- `documents/` — 한영 등록 문구, 개인정보처리방침, 심사 메모, 체크리스트
- `00_READ_ME_FIRST.md` — Chrome 등록 순서

## 2. NAVER Whale Store

`02_NAVER_Whale_Store/`

- `package/` — Whale Store에 업로드할 ZIP 1개
- `images/` — 128px 아이콘과 한국어·영어 스크린샷
- `documents/` — 한영 등록 문구, 개인정보처리방침, 심사 메모, 체크리스트
- `00_READ_ME_FIRST.md` — Whale 등록 순서

## 3. Microsoft Edge Add-ons

`03_Microsoft_Edge_Add-ons/`

- `package/` — Microsoft Partner Center에 업로드할 ZIP 1개
- `images/` — 128px·300px 아이콘, 한국어·영어 스크린샷과 프로모션 타일
- `documents/` — 한영 등록 문구, 개인정보처리방침, 심사 메모, 체크리스트
- `00_READ_ME_FIRST.md` — Edge 등록 순서

## 제출자가 입력할 항목

- 공개 지원 이메일
- 공개 개인정보처리방침 URL
- 각 스토어의 게시자·계정 정보와 공개 범위

`store-assets/`는 이미지 제작용 원본이고, `store-submissions/`는 내부 생성 원본입니다. 실제 등록에는 이 `스토어별_최종제출/` 폴더만 사용하면 됩니다.
